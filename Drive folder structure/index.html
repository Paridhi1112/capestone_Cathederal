File Naming Convention Examples

Board Minutes
❌ Bad:
Minutes March.doc
✅ Good:
2026-03-15_Board_Meeting_Minutes_v1_FINAL.pdf
Service Bulletin
❌ Bad:
Easter 2026.pdf
✅ Good:
2026-03-31_Service_Bulletin_Easter_Sunday.pdf
Budget
❌ Bad:
Budget FINAL!!!.xlsx
✅ Good:
2026_Annual_Budget_v4_FINAL.xlsx
Grant Application
❌ Bad:
Grant Application (March) - FINAL!!!.docx
✅ Good:

donor thank you.docx

Photo
❌ Bad:

Key Naming Rules:

  • Date First: YYYY-MM-DD format ensures chronological sorting
  • Underscores: Use _ instead of spaces for compatibility
  • Title_Case: Capitalize Each Word for readability
  • No Special Characters: Avoid ! @ # $ % ^ & * ( ) [ ] etc.
  • Version Numbers: Use v1, v2, v3 or DRAFT, FINAL
  • Under 100 Characters: Keep filenames concise but descriptive
